# gour.me
あなたに合ったグルメをレコメンドします.

### Setup

```bash
$ make install
```

### Starting develop server

```bash
$ php artisan serve
$ npm run dev
```

### Rebuilding DB

```bash
$ php artisan migrate:fresh --seed
```
